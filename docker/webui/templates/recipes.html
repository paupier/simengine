Stop Conditions
  • Quantity — stop after producing N parts (batch mode)
  • Duration — stop after N seconds (time-box mode)
  • Max duration — safety timeout for quantity-based stops
Changeover Distributions
  • constant — fixed value
  • normal — mean + std
  • lognormal — mean + std
  • exponential — mean only
  • uniform — min + max
  • weibull — shape + scale
Machine Overrides

Override per-segment machine settings. Match by machine name. Supported: cycle_time, target_ppm, defect_rate.
