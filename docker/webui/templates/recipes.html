Recipe YAML Reference
Structure
name: "Recipe Name" description: "What this recipe does" base_scenario: balanced_line segments: - name: "Segment 1" quantity: 500 # stop after N parts max_duration: 18000 # safety timeout (s) overrides: # optional machines: - name: M1 cycle_time: 10 defect_rate: 0.02 changeover: # before next segment target: 300 distribution: lognormal mean: 300 std: 60 - name: "Segment 2" duration: 7200 # stop after N seconds
Stop Conditions
  • quantity — stop after producing N parts (batch mode)
  • duration — stop after N seconds (time-box mode)
  • max_duration — safety timeout for quantity-based stops
Changeover Distributions
  • constant — value: 10
  • normal — mean: 300, std: 60
  • lognormal — mean: 300, std: 60
  • exponential — mean: 120
  • uniform — min: 60, max: 180
  • weibull — shape: 2, scale: 300
Machine Overrides

Override per-segment machine settings under overrides.machines. Match by name. Supported fields: cycle_time, target_ppm, defect_rate.
